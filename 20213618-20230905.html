*简介
        Git Bash 在 Windows 上模拟 bash 环境，允许用户在 Windows 操作系统上使用 Bash shell 和大多数标准 Unix 命令。用户可以通过在 Git Bash 中运行命令来与存储库和 Git 元素进行交互。
第一节课主要下载了git cmd ；git gui；git bash，主要学习了一些简单的git bash命令如创建个人文件
夹，对文件文件夹反复进行拷贝删除创建展开等进行了简单的操作。

*主要操作步骤
下载安装git bash
学习简单的目录、文件操作指令：
cd ( change directory ) 切换目标
如：进入D目录：cd /d；进入D目录下的子目录lk：cd /D/lk；
ls ( list ) 查看当前目录下的内容
ls -al 包括隐藏文件和以 . 开头的文件
mkdir ( make directory ) 创建目录
rmdir ( remove directory )删除文件夹；或者rm -rf（目录可以有文件）
touch 创建文件：touch lk.txt
cat 查看文件内容 :cat lk.txt
cp ( copy ) 复制文件
cp -r A/* B 复制 A 文件夹下的所有子文件、目录到 B 文件夹
rm ( remove ) 删除文件
mv ( move ) 移动文件或重命名
